What is Superposition?

Neural networks can represent more features than dimensions by using non-orthogonal directions. When features are sparse, interference is rare enough that overlapping representations work.

n_features >> n_hidden (if features are sparse)
The Superposition Hypothesis

Core Components

Toy Models

Simple bottleneck autoencoders that exhibit superposition. Train models with configurable sparsity and importance.

Interference Analysis

Measure pairwise interference between features. Compute I[i,j] = cos^2(w_i, w_j).

Effective Dimensionality

How many "real" dimensions each feature uses. d_eff < 1 means sharing space with others.

Linear Probing

Test feature decodability. Measure polysemanticity - neurons responding to multiple features.

Key Findings

From "Toy Models of Superposition" (Anthropic)

  • Sparsity enables superposition - More sparse features = more can be packed
  • Importance matters - Important features get more orthogonal directions
  • Phase transitions - Sharp transitions between mono/polysemantic representations
  • Geometric structure - Features form regular polytopes in weight space
